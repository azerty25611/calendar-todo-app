このプロジェクトは曜日ごとにやることを簡単に設定できるカレンダー型のウェブアプリケーションです。

## 重要機能
カレンダー
曜日別に入ってやることを追加、編集、削除

## 使用技術
- HTML
- CSS
- JavaScript
- LocalStorage API
- Git, GitHub

## 実行方法
１．このリポジトリをクローンあるいはダウンロードします。
２．index.htmlファイルをブラウザから開けるとすぐ実行します。
３．あるいは下の配布リンクから直接使用してください。
